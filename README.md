## Run Locally

Clone the project

```bash
  git clone https://github.com/willliamdev/trilha-ignite.git
```

Go to the project directory

```bash
  cd trilha-ignite
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
